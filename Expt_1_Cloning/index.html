Shallow Copy(t1 & t3):
 Two objects t1 & t3 are pointing at same memory location i.e. Memory Location 1. So, when member variable changes, it is reflected in both the objects.

Deep Copy(t1 & t2):
 Two objects t1 & t2 are pointing at two different memory location i.e. Memory Location 1 & Memory Location 2. So, when member variable of one location changes, it does not affect on other object.
